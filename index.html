About Text Update App

Text Update App is a desktop application that helps you improve your writing by checking grammar and rephrasing text using AI.

Features

  • Grammar Check: Identifies and corrects grammar errors in your text.
  • Text Rephrasing: Rewrites your text to improve clarity and style.

How to Use

  1. Select text in any application (browser, word processor, etc.)
  2. Use the keyboard shortcuts to check grammar or rephrase
  3. The app will process the text and replace your selection

Troubleshooting

  • Shortcuts not working: Make sure no other application is using the same shortcuts.
  • Text not being replaced: Some applications may restrict clipboard access. Try using the app with different applications.
  • API errors: Check your API keys and internet connection in the Settings.
