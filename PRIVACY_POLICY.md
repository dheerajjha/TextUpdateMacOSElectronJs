# Privacy Policy for Grammar Ji

**Last Updated: [Current Date]**

## Overview
Grammar Ji ("we", "our", or "the app") is committed to protecting your privacy. This Privacy Policy explains how we handle information when you use our application.

## Information We Don't Collect
Grammar Ji does NOT collect, store, or transmit any personal information including:
- Names, email addresses, or contact information
- Account credentials (no accounts required)
- Usage analytics or statistics
- Device identifiers
- Location data
- Any other personally identifiable information

## How Grammar Ji Works

### Text Processing
When you use Grammar Ji to check grammar, rephrase, summarize, or translate text:
1. You select text in any application
2. The selected text is sent to our secure API endpoint for processing
3. The text is processed in real-time
4. The improved text is returned to you
5. **The text is immediately discarded and not stored anywhere**

### Data Transmission
- Text is transmitted over encrypted HTTPS connections
- Processing occurs on secure Azure OpenAI servers
- No logs or records of your text are maintained
- No user identification or session tracking is performed

## Third-Party Services

### API Provider
Grammar Ji uses Azure OpenAI (Microsoft) for text processing:
- Text is processed according to Microsoft Azure's privacy standards
- Microsoft does not use customer data to train models
- Data is not shared with other third parties
- Learn more: https://privacy.microsoft.com/

## Permissions Required

Grammar Ji requests the following macOS permissions:

### Accessibility Access
- **Purpose**: To read selected text from any application
- **Usage**: Only activated when you trigger a Grammar Ji shortcut
- **Scope**: Limited to text you explicitly select

### Network Access
- **Purpose**: To send text to our API for processing
- **Usage**: Only when you use a Grammar Ji feature
- **Data**: Only the selected text, nothing else

### Clipboard Access
- **Purpose**: To replace selected text with improved version
- **Usage**: Only when processing is complete
- **Scope**: Limited to the text you selected

## Data Security
- All API communications use HTTPS encryption
- No data is stored on our servers
- No data is stored locally beyond app settings (feature toggles, shortcuts)
- Your text is processed in transit only

## Children's Privacy
Grammar Ji does not knowingly collect any information from children under 13 years of age. The app does not require any personal information and can be used safely by users of all ages.

## Your Rights
Since we don't collect any personal data:
- There is no data to access, correct, or delete
- There are no user profiles or accounts
- You can uninstall the app at any time with no data remaining

## Local Storage
Grammar Ji only stores locally on your device:
- App preferences (feature toggles, keyboard shortcuts)
- Theme preference (light/dark mode)
- Usage statistics (counts only, no text content)

This data:
- Stays on your device only
- Is not transmitted anywhere
- Can be reset via app settings
- Is deleted when you uninstall the app
