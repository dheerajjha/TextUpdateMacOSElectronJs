About Text Update App

        
Text Update App is a desktop application that helps you improve your writing by checking grammar and rephrasing text using AI.

        
        
Features

        
    
          
  • Grammar Check: Corrects grammar errors in your text
    • 
          
  • Text Rephrasing: Makes your text more clear and concise
    • 
          
  • Global Shortcuts: Works in any application
    • 
        

        
        
How to Use

        
    
          
  1. Select any text in any application
    2. 
          
  2. Press Command+Shift+G to check grammar
    3. 
          
  3. Press Command+Shift+R to rephrase text
    4. 
          
  4. The processed text will replace your selection
    5. 
        

        
        
Troubleshooting

        
If the app is not working correctly:

        
    
          
  • Make sure you have selected text before using the shortcuts
    • 
          
  • Check your API settings in the Settings menu
    • 
          
  • Restart the application
    •
